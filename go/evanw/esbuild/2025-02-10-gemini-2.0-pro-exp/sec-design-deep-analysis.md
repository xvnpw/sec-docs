While it interacts with external systems (npm for dependencies), the core bundling logic is self-contained. This is a key difference from tools like Webpack, which rely heavily on plugins and loaders.

    *   **Resolver:**  This component is responsible for resolving module dependencies.  It likely implements Node.js-style module resolution (looking in `node_modules`, handling `package.json`, etc.).  It also likely handles path aliasing and other configuration options related to module resolution.
